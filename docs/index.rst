Welcome to HatosGanaderos's documentation!
==========================================

HatosGanaderos es un sistema web que ayuda al ganadero o entidad ganadera con el control y organización de reproducción, alimentación, sanidad y producción, con la clara finalidad de prestar atención a las necesidades de los animales vacunos. 

Qué es HatosGanaderos?
----------------------
HatosGanaderos es un sistema web de código abierto realizado en tecnologías como: python, django, html, css, NodeJs, SPADE, css, postgreSQL entre otras. El software se encuentra en su primera version (1.0.0) y te ayuda a mantener los registros de tus ganados, esperma, alimentos, desparasitadores, vacunas, registros de producción de leche, registro de técnicos, servicio de mensajería entre técnicos y registro de técnicos; facilitando el proceso de las entidades ganderas.

HatosGanaderos es una excelente opción para tu ganadería, deberías probarla!
